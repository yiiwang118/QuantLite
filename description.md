# Quant Lite — 项目简介

> 已生成图标：`icon.svg`（矢量源）/ `icon.png`（1024×1024）/ `icon-512.png` / `icon-144.png`

## 简短描述（约 110 字，可直接用于小程序提交）

Quant Lite 是一款面向团队的多市场数据研究工具。内置 AI 助手 LiteAI，用户可用自然语言描述需求，系统自动生成基于公开历史行情的分析脚本与对照报告。支持时序计算、横截面运算、批量回看、团队共享与协作。所有数据均来自公开市场行情，仅用于内部研究、教学与算法学习交流，不构成任何投资建议或交易决策依据。

## 申请小程序的几点提醒

- **主营类目**建议选 **「工具 → 效率」** 或 **「教育 → 学习教育」**，不要选金融类（避开监管类目，审核会更顺）
- 介绍文案 / 截图描述统一避开：**投资、推荐、收益、买卖、交易、信号、选股、配资、理财**
- 可用的中性词：**研究、分析、回看、对照、数据、学习、协作、脚本、参数**
- 提交时强调：**仅供学习与研究**、**数据来自公开渠道**、**不提供投资建议**
